## 📖 关于项目

**小书袋** 是我在学习使用 AI 编程时做的一个个人小工具 🛠️

作为一个小说爱好者，我经常想记录下读过的书、当时的感受，还有自己的评分。但是用笔记软件太零散，用 Excel 又太严肃……所以就有了这个小书袋～ 📦

不需要下载 App，不需要注册账号，打开就能用，数据只存在你自己的浏览器里，安心又方便 ✨

## ✨ 功能特点

| 功能 | 说明 |
|------|------|
| 📝 | **记录阅读** — 记录书名、评分、书评、阅读日期，还能添加自定义标签 |

| 📤 | **数据导出** — 一键导出所有记录，格式与录入格式一致，方便备份 |

## 💡 使用小贴士

1. **首次使用**：会有一个初始化弹窗，可以批量导入之前看过的书，格式：`日期/书名/评分/书评`（一行一本）。也可以直接跳过，之后一本一本添加～
2. **评分格式**：支持 1-5 分，可以带小数（比如 4.3 分），卡片上会显示对应的星星数（四舍五入）。
3. **标签功能**：可以创建自定义标签，给书分类（比如“科幻”、“言情”、“神作”）。点击标签就可以只显示该标签下的书。
4. **编辑/删除**：双击卡片或点击卡片右下角的铅笔图标，就可以编辑或删除记录。
5. **数据导出**：点击“导出”按钮，可以一键复制所有阅读记录（按照当前排序），方便备份或迁移。
6. **数据保存**：所有数据都存在浏览器的 localStorage 里，清除浏览器缓存会导致数据丢失，记得定期导出备份哦～

## ❓ 常见问题

**Q: 数据安全吗？**  
A: 安全！数据只存在你的浏览器本地，不会上传到任何服务器。

**Q: 换设备/浏览器后数据还在吗？**  
A: 不在哦～localStorage 是绑定特定浏览器的，换设备需要重新录入。  
可以用“导出”功能先备份，再在新设备上通过初始化录入恢复。

**Q: 如何清除所有数据？**  
A: 在浏览器开发者工具里清除 localStorage，或者手动删除所有记录即可。

**Q: 标签可以删除吗？**  
A: 目前暂不支持删除标签（因为标签会关联到书籍），你可以通过编辑卡片移除标签，让标签自然“失效”。
